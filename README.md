# Jane Kissington — Portfolio & Services

Professional portfolio and services landing page.

## Live Site

Deployed via GitHub Pages.

## What's Included

- **Services**: Website audit service ($149+) and custom automation ($99/mo+)
- **Products**: Next.js SaaS Starter Kit ($49) and SiteAudit CLI ($29)
- **About**: Bio, tech stack, key stats
- **Contact**: Email, GitHub, contact form (mailto-based)

## Technical Details

- Pure HTML/CSS/JS — no frameworks, no build step
- Single `index.html` file (~20KB)
- Dark theme, fully responsive (mobile-first)
- Smooth scroll navigation with active link tracking
- Scroll-triggered fade-in animations
- SEO meta tags and Open Graph support
- Inline SVG favicon

## Deployment

Push to a GitHub repo and enable GitHub Pages from Settings > Pages > Deploy from branch (main).

```bash
git init
git add .
git commit -m "Initial portfolio site"
git remote add origin https://github.com/janekissington/janekissington.github.io.git
git push -u origin main
```

The site will be live at `https://janekissington.github.io`.

## Updating

Edit `index.html` directly. No build step required.
